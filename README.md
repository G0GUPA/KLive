# KLive
KLive는 라이브 방송을 제공하는 사이트 및 앱을 크롤링하는 python 코드이다.

이를 사용한 KODI addon과 PLEX channel plugin이 포함되어져 있으며, 네트워크 재생을 지원하는 플레이어에서 재생할 수 있도록 M3U, EPG를 생성하는 Klive Server가 있다.

KLive Server와 tvheadend 를 이용하면 녹화도 가능하다.
